# Graded Assignment 3 — Systems Engineering: Data, Security, Architecture & Operating Systems

**System Name:** CampusConnect Backend  
**Database Engine:** SQLite 3.45  
**Programming Language:** Python 3.10+  

## Project Overview
This repository contains the complete implementation for the 4-part Systems Engineering assessment for CampusConnect.

- **Part 1: Relational Database Design, Querying, Indexing & Transactions** (`/Part1_Database`)
- **Part 2: Cryptographic Protocol Implementation & Network Security Threat Analysis** (`/Part2_Cryptography`)
- **Part 3: Software Architecture, Low-Level Design & Cloud Deployment Strategy** (`/Part3_Architecture`)
- **Part 4: CPU Scheduling Simulation, Synchronization & Deadlock Analysis** (`/Part4_OS`)

Each directory is standalone and contains its own dedicated runnable scripts and documentation.
